# SweetCatch

## Objective

Catch as much fruit as you can to earn as many points as you can! Avoid the bad fruit so you don't lose any lives!

*Note: This game is compatible with XCode and IOS devices.*

## Files Contain
-*Basket*
-*Different pieces of fruit*
-*A background*
-*Sound effects*
-*Background Music*
-*A bad fruit*
